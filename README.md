# asRP-python

ERP (Enterprise Resource Planning) en Python con microservicios, contenedores y seguridad.

## Estructura
- `services/`: microservicios (cada uno con su propia base de datos)
- `libs/`: librerías compartidas (mínimas y versionadas)
- `infra/`: infraestructura local/cloud (Docker Compose, etc.)
- `docs/`: documentación (Markdown)
